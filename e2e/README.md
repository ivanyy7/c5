# E2E тесты калькулятора с Playwright

Этот каталог содержит End-to-End (E2E) тесты для проверки работы калькулятора в браузере.

## Установка

1. Установите зависимости (если еще не установлены):
```powershell
npm install
```

2. Установите браузеры для Playwright:
```powershell
npx playwright install
```

## Запуск тестов

### Базовый запуск (headless режим)
```powershell
npm run test:e2e
```

### Запуск с UI интерфейсом
```powershell
npm run test:e2e:ui
```

### Запуск в видимом режиме (headed)
```powershell
npm run test:e2e:headed
```

### Запуск в режиме отладки
```powershell
npm run test:e2e:debug
```

### Запуск конкретного теста
```powershell
npx playwright test calculator.spec.ts
```

### Запуск тестов для конкретного браузера
```powershell
npx playwright test --project=chromium
npx playwright test --project=firefox
npx playwright test --project=webkit
```

## Что тестируется

### ✅ Ввод чисел
- Ввод одиночных и многозначных чисел
- Ввод десятичных чисел
- Замена нуля при вводе первого числа
- Предотвращение множественных точек

### ✅ Математические операции
- Сложение (+)
- Вычитание (-)
- Умножение (×)
- Деление (/)
- Сложные выражения с несколькими операциями

### ✅ Отображение результатов
- Корректное отображение результатов вычислений
- Обработка ошибок (деление на ноль)
- Отображение десятичных чисел

### ✅ Специальные функции
- Кнопка очистки (C)
- Кнопка удаления последнего символа (⌫)
- Повторное использование результата

### ✅ Ввод с клавиатуры
- Ввод чисел с клавиатуры
- Выполнение операций с клавиатуры
- Очистка по Escape
- Удаление по Backspace

## Структура тестов

- `calculator.spec.ts` - основной файл с E2E тестами калькулятора

## Отчеты

После запуска тестов отчет будет доступен по адресу:
```
playwright-report/index.html
```

Для просмотра отчета:
```powershell
npx playwright show-report
```
